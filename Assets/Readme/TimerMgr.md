# TimerMgr - 高精度定时器

## 1. 概念 (Concept)
`TimerMgr` 是游戏逻辑的时间驱动引擎。它解决了直接使用 `Coroutine` 导致的逻辑散乱、难以管理以及性能损耗问题。

## 2. 逻辑 (Logic)
*   **双轴计时**：
    *   提供 `TimerMgr` (基于现实时间 `Time.deltaTime`)。
    *   提供 `FrameTimerMgr` (基于逻辑帧数)。
*   **SafeMap 管理**：
    *   内部使用 `SafeMap` 存储定时器，支持在定时器回调中安全地删除自己或其他定时器，防止集合修改异常。
*   **生命周期绑定**：
    *   UI 或 Entity 在销毁时，可以一键取消其注册的所有定时器，防止闭包引起的内存泄漏或逻辑空指针。

## 3. 技术流 (Applied Technology)
*   **句柄模式**：返回唯一的 `int` 类型 `guid`，用于追踪和回收。
*   **TimeScale 支持**：支持对特定定时器进行时间缩放（如子弹时间），不影响全局计时。
