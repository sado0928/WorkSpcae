# EffectMgr - 粒子与特效控制

## 1. 概念 (Concept)
`EffectMgr` 负责游戏中所有视觉表现特效的生命周期。它在 `PoolMgr` 的基础上增加了针对特效的画质适配与数量限制。

## 2. 逻辑 (Logic)
*   **画质适配 (EffectConfig)**：
    *   支持定义高/中/低三档特效配置。
    *   切换配置时，会自动遍历所有活跃特效并应用对应的参数（如粒子数量缩减、光效开关）。
*   **数量限制 (Limit Mechanism)**：
    *   当屏幕特效过多时，会自动根据“先进先出”或优先级原则清理多余特效，确保帧率稳定。
*   **自动回收**：
    *   特效播放结束后，通过其绑定的 `EffectBase` 逻辑自动归还给 `PoolMgr`，实现自动化闭环。

## 3. 技术流 (Applied Technology)
*   **特效句柄 (EffectHandle)**：逻辑层通过句柄控制特效的位移、缩放与强制销毁。
*   **生命周期钩子**：利用粒子系统的 `main.duration` 自动计算销毁时间。
